## 🛠️ **Getting Started**

### **Step 1: Clone the Repository**
```bash
git clone https://github.com/Komal-Khan-PhD/foundations-of-ml.git
cd foundations-of-ml
```

### **Step 2: Install Requirements**
```bash
pip install -r requirements.txt
```

### **Step 3: Launch Jupyter**
```bash
jupyter notebook
```
